# HRMS Frontend

React + Vite frontend for the HRMS application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional):
```env
VITE_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Redux Toolkit
- React Router
- Tailwind CSS
- Axios
